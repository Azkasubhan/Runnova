# 📊 Predictive Maintenance System for SMEs

A scalable system designed to help Small and Medium Enterprises (SMEs) reduce costly downtime through **Remaining Useful Life (RUL) estimation**, **anomaly/change-point detection**, and an intuitive **machine health dashboard**.

---

## ✨ Overview

Many SMEs still rely on aging machines that frequently break down. This system transforms maintenance from **reactive → predictive**, enabling better planning, reduced downtime, and lower operational costs.

---

## 🚩 Problem

| Issue | Impact |
|------|--------|
| Old machines frequently fail | High risk of unexpected downtime |
| Repairs only after failure | Long recovery time (up to 1 week) |
| Production stops completely | Business operations disrupted |
| Workers cannot work | Loss of income |
| Unscheduled maintenance | Inefficient planning |
| Sudden repair costs | Increased operational expenses |

---

## 💡 Solution

This system helps managers:

- Identify **machines that are likely to fail soon**
- Monitor machine condition in real-time
- Make **data-driven maintenance decisions**
- Use insights through a **simple, non-technical dashboard**

---

## ⚙️ System Implementation

### 📥 Input Data (Sensor-Based)

- Vibration
- Temperature
- Load
- Pressure *(optional)*
- Speed
- Electrical Current

### ⚡ Core Processing

- Machine health evaluation
- Remaining Useful Life (RUL) prediction
- Anomaly detection
- Change-point detection

---

## 📊 Dashboard Features

| Feature | Description |
|--------|------------|
| Machine Health Score | Overall condition of each machine |
| Predicted RUL | Estimated remaining operational time |
| Anomaly Alerts | Early warnings of abnormal behavior |
| Trend Over Time | Performance visualization |
| Maintenance Priority Ranking | Which machines to fix first |

---

## 🔑 Key Features

- **Real-Time Machine Health Monitoring**  
- **Remaining Useful Life Prediction (RUL)**  
- **Early Degradation Detection**  
- **Maintenance Priority Recommendation**  
- **Interpretable Alerts for Non-Technical Users**

---

## ## 🎯 Objective

To help SMEs:

- Reduce unexpected downtime  
- Minimize maintenance costs  
- Improve operational efficiency  
- Enable smarter maintenance planning
